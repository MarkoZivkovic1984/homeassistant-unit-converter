# homeassistant-unit-converter

Homeassistant headless cli unit converter

Unit Converter

A Python application that converts between multiple units of distance, mass, and temperature Pint.

Features:

Converts units like meters ↔ miles, °C ↔ °F, kilograms ↔ pounds.

Automatically handles unit conversion logic.

Usage:

Clone the repository:

git clone https://github.com/MarkoZivkovic1984/homeassistant-unit-converter.git

Used Pyscript inside the Home Assistant

import Pint as requierement

add yaml configurations that are provided

Example:

Enter a value in Količina.

Select input unit Iz Mere.

Select output unit U Meru.

Click Izračunaj to see the converted value.

Dependencies

Python 3.x

pint
